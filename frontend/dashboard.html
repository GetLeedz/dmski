Was muss ich nach dem Einloggen tun?

Je nach Ihrer Rolle als File Owner oder File Reviewer sind unterschiedliche Schritte nötig.

Direkt zum Fall — in drei Schritten

Als File Reviewer (Gutachter, Anwalt, Therapeut o. ä.) ist Ihr Zugang bereits eingerichtet. Sie müssen keinen neuen Fall anlegen. So gehen Sie vor:

  1. ① Wählen Sie unter «Bestehendes Dossier öffnen» das Ihnen zugewiesene Dossier aus der Dropdown-Liste. Dossiernummer und Fallname erhalten Sie vorab per E-Mail vom File Owner.
  2. ② Nach der Auswahl werden Sie sofort und automatisch zur vollständigen Aktenübersicht weitergeleitet — mit allen Files, KI-Analysen und dem forensischen Bericht.
  3. ③ Sie können Files einzeln herunterladen oder den gesamten KI-Report als PDF exportieren. Das Hochladen neuer Files ist ausschliesslich dem File Owner vorbehalten.

Vollständiger Arbeitsablauf — File Owner

  1. 1
    Neues Dossier anlegen oder bestehendes öffnen

    Neu: Formular links ausfüllen (Fallname, Personen, Ort) und auf «Fall anlegen» klicken. Bestehend: Dossier direkt aus der Dropdown-Liste rechts auswählen.

  2. 2
    Files hochladen nur File Owner

    Nach dem Öffnen des Dossiers erscheint die Upload-Seite. Laden Sie Akten, Gutachten, Berichte oder Beweismittel hoch (PDF, JPG, PNG) — einzeln oder per Drag & Drop gleichzeitig.

  3. 3
    KI-Analyse läuft automatisch

    Nach jedem Upload startet die KI-Analyse ohne weiteres Zutun. Sie erkennt Beteiligte, bewertet Erwähnungen (positiv / negativ) und erstellt eine forensische Einschätzung je File sowie eine Gesamtbewertung des Dossiers.

  4. 4
    Dossierübersicht & Bericht einsehen

    Klicken Sie in der Sidebar auf «Fall-Analyse», um alle Files, Einzelanalysen, den KI-Gesamtbericht und die Taktikanalyse einzusehen. Von dort aus können Sie Files herunterladen oder den PDF-KI-Report exportieren.

Rollen: Der File Owner erstellt Fälle und lädt Files hoch. File Reviewer (Gutachter, Anwalt, Therapeut …) haben Lesezugriff und können Files und den KI-Report herunterladen. Zugangsdaten erhalten File Reviewer per E-Mail vom File Owner.
